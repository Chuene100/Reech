## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```
- expo
  ```sh
  npm install -g expo-cli
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/reechdevelopers/reech-mac-mob-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run application
   ```js
   expo start
   ```
4. Run android
   ```js
   expo start android
   ```
5. Run ios
   ```js
   expo start ios
   ```

### Building an (Android) APK

1. Put the credetials.json file at the root of the project
2. Create a "`plugins`" folder at the root and add the network_security_config.xml and trust-local-certs.js files inside it.
3. Add "`./plugins/trust-local-certs.js`" to the array of plugins:[...] inside `app.json` file
4. Add `"credentialsSource": "local"`, under each profile (preview, development) inside the `eas.json` file
5. Add keystores/ to the `.gitignore` file
6. Create a keystores folder at the root of the project
7. Add the `reecheble_tech.keystore` file inside the keystores/ folder
8. Run the command 
   ```js
   $ npx expo prebuild
   ```
9. Delete the "`android`" and the "`ios`" folders that are generated by that command
10. Run the command
    ```js
    $ npx expo prebuild --no-install
    ```
11. Again, delete the "`android`" and the "`ios`" folders that are generated by that command
12. You can now build or run the project with `npm start`
13. Run the command
    ```js
    $ npm run build-android
    ```
